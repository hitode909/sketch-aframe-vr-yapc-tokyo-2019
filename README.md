# VR YAPC Tokyo 2019

Unofficial VR version website of [YAPC::Tokyo 2019](https://yapcjapan.org/2019tokyo/)

## HOW TO RUN

```
python -m SimpleHTTPServer 3100 & ngrok http 3100
open http://localhost:3100
```
